# Opfermaterial

Das Alu-Rohmaterial ist **immer** auf ein Opfermaterial zu spannen, um die Nutenplatte zu schützen. Dafür eignen sich dünne Siebdruck- oder Multiplexplatten, die auf das Maß des Werkstücks gesägt wurden.
